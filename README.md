# securisation-risk-analysis

## Overview
This project is an interactive Power BI dashboard developed for analysing securitisation risk using sample auto loan portfolio data. The dashboard provides key insights into portfolio performance, delinquency trends, vintage analysis, and monthly loss metrics to support financial risk assessment and decision-making.

---
